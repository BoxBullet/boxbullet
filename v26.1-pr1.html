THE END
  • Added a new dimension, The End.
  • Added 2 new biomes, The End and Void.
  • Added a new terrain generation system for The End, using the biomes The End and The Void.
  • Added a forcefield around The End's biome, where The End intersects The Void, damaging you and knocking you back onto The End biome.
  • Added a new boss, The Apex, having multiple stages (WILL BE REWORKED!)
  • Added a black portal in the center of The End's island to teleport you back to The Overworld.
  • Added a new item, the brimstone pearl, crafted with 1 vortex pearl and 1 brimstone powder. When right clicked, it puts you through an animation leading to the end dimension.
GAME SAVES
  • The game now saves all player, entity, and basic world data in a singleplayer_save.json every 30 seconds or when the player dies or disconnects.
  • World and player data now gets loaded and applied upon joining single player. If the file doesn't exist, it simply creates the world.
  • The player's inventory now gets reset upon death and is given an iron sword. Their health also gets reset to 100.

NEW MOBS
  • The Vortex:
    • A mob shooting its new `vortex/gust.png` which deals very little damage. It has a small chance to replace raidshroomers during a raid. It also drops a new `Vortex Pearl`.
  • The Apex:
    • A mob with multiple stages, changing its appearance and AI on each one. It drops nothing, and only spawns in The End as a final boss. (WILL BE REWORKED)
CRAFTING AND INVENTORY
  • Below the main inventory there is now a place for crafting. The current crafting recepies are:
    • Brimstone Powder + Vortex Pearl = Brimstone Pearl
    • Any Wood Weapon + Rock = Stone Variant
    • Any Stone Weapon + Copper = Copper Variant
    • Any Copper Weapon + Iron = Iron Variant
    • Any Iron Weapon + Diamond = Diamond Variant
    • Any Diamond Weapon + Brimstone Powder = Brimstone Variant
  • Added right-click to pick up half the stack in the inventory.
  • Completed raid outposts can now be RIGHT CLICKED to open their chest. Their current loot table is:
    • All basic ores (Not including brimstone powder)
    • Raidshrooms
    • Vortex Pearls
    • All wood weapons
    • All stone weapons
  • Updated the inventory from byte-based items to string-based items.
