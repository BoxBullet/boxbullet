Bulletbox 26.1.1-01c
Alpha Unreleased
Core Updates
  • Asset Management System: Introduced a centralized loader for HD textures to optimize VRAM usage and enable high-fidelity sprite rendering.
  • Player Animation System: Implemented a 2-frame, 4 FPS animation for player characters using a 64x128px spritesheet.
  • Iterative Asset Pipeline: Established a workflow for upscaling 16x16 pixel art prototypes into HD entity textures.
  • Asset Loading Diagnostics: Added console logging to verify successful texture loading and assist in debugging asset path issues.
  • Debug Utility System: Added a toggleable hitbox visualization tool (Grave + H) for development and hit-reg testing.
  • Options System: Introduced a dedicated Options screen featuring a persistent FOV/Zoom slider to customize visual perspective.
  • Namespace Resolution: Ensured proper access to namespaced classes (e.g., Settings) from global namespace files.
  • Camera Zoom Abstraction: Implemented a dedicated Zoom property on CameraManager to control the camera's field of view, resolving direct access errors.
